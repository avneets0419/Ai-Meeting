## ğŸ§  How It Works

1. **Upload a recording** from your dashboard.  
2. Backend uploads file to **AssemblyAI** â†’ receives `upload_url`.  
3. A **transcription job** starts automatically.  
4. MeetWise periodically checks the job status via `/status/:id`.  
5. Once complete, it retrieves:
   - Transcript text
   - Summary (auto-generated by AssemblyAI)
   - Action items
6. Tasks appear on your **Kanban board** for easy tracking.
